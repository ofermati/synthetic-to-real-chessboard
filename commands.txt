1. Create a virtual environment (one-time setup)
windows:
python -m venv .venv

mac:
python3 -m venv .venv

2. Activate the virtual environment

Windows - PowerShell (most common in VS Code):
.\.venv\Scripts\Activate.ps1

Windows - Command Prompt (CMD):
.\.venv\Scripts\activate.bat

Windows - Git Bash:
source .venv/Scripts/activate

macOS / Linux:
source .venv/bin/activate


3. Install project dependencies
pip install -r requirements.txt

4. Verifying everything is working
4.1 Check Python path:
    windows:
    where python  

    macOS/Linux:
    which python

4.2 Check installed packages:
    pip list




git commands:
git commit --amend --no-edit להוסיף משהו לcommit שהעלתי
